# TypeScript SDK

The Echo TypeScript SDK provides a fully-typed interface for interacting with the Echo platform. It includes client initialization, authentication, and all core resources.

## Installation

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">

  </Tab>
  <Tab value="pnpm">

## Quick Start

```typescript
import { EchoClient } from '@merit-systems/echo-typescript-sdk';

const client = new EchoClient({
  apiKey: 'your-api-key'
});

// Use the client resources
const apps = await client.apps.listEchoApps();
const balance = await client.balance.getBalance();
const models = await client.models.getSupportedModels();
```

## API Resources

The EchoClient provides access to the following resources:

### client.apps - App Management

Manage your Echo applications.

**Available Methods:**
- `client.apps.listEchoApps(): Promise<EchoApp[]>` - List all Echo apps
- `client.apps.getEchoApp(appId: string): Promise<EchoApp>` - Get specific app
- `client.apps.getAppUrl(appId: string): string` - Get app URL

**Related Types:**

```typescript
import type { EchoApp } from '@merit-systems/echo-typescript-sdk';
```

### client.models - Model Information

Get information about supported LLM models.
