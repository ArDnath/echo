# TypeScript SDK

The Echo TypeScript SDK provides a fully-typed interface for interacting with the Echo platform. It includes client initialization, authentication, and all core resources.

## Installation

```bash
npm install @merit-systems/echo-typescript-sdk
```

## Quick Start

```typescript
import { EchoClient } from '@merit-systems/echo-typescript-sdk';

const client = new EchoClient({
  apiKey: 'your-api-key'
});

// Use the client
const apps = await client.apps.list();
```
