# What is Echo?

Echo is a service and set of client libraries that provides authentication and 
usage-based billing for expensive AI resources. Install in your app and ship your 
monetized AI application without the risk massive OpenAI bills.

Select your app's architecture to get started or view the example application 
or repository with the buttons below.
